Big Data Analytics: Extraction of Most Popular Dishes from Restaurants Based on Yelp's User Reviews

20 Nov 2015

This paper summarizes the analytics done on user reviews to understand "must have” dishes in a particular restaurant. An anlytics was performed on User data, Restaurant data, User reviews data and Restaurant menus data that we obtained from Yelp and Foursuare. We used few Sentiment Analysis tools to process user reviews and calculate average scores for each of the dishes. We have suggested "must have” dishes from a particular restaurant depending on these scores..

Read acknowledgments for gmail - Get notified as soon as the recipient reads your email

16 Jan 2014

Gmail users no longer have to click the "Display images below" link to view the images embedded in the email. Sheer convienience, right :) Until a couple months back, gmail used to give users a "Display images below" option. The inline images were displayed only after user clicked on the link. This was to protect users from unsafe contents and websites. In an announcement previous month, google announced that going forward all images will be displayed inline without requiring any action from the user.
